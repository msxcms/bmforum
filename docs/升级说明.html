从 6.0 升级安装说明
返回顶部
注意 改动较大，建议先测试灰度升级后再进行全量升级！
请注意上传文件的完整性


  1. 升级途中请务必关闭论坛。建议先备份一下论坛数据。
  2. 上传、变更下列文件或目录：
    1. 上传下列文件
      • 压缩包中 bmb 目录下的所有文件（1、不包括 bmb 目录里面的文件夹；2、不要上传install.php 和 install目录）
      • 目录:lang
      • 目录:newtem
      • 目录:plugins
      • 目录:include
      • 目录:upgrade
      • 目录:editor
      • 目录:images

  3. 打开浏览器，进入论坛网址目录下的 upgrade （例如：http://www.yourname.com/bbs/upgrade/ ），按照程序步骤操作。
  4. 管理中心，开始菜单中调整论坛数据，刷新风格缓存
  5. 常规选项、用户组内有新设置，请务必进入管理中心设置（尤其是需要在常规选项内设置默认表情包）
  6. 如果您制作了自己的语言包，请访问官方 Blog 下载升级手册
  7. 论坛旧表情已经在刚才转移到了 images/oldface，如果需要，请参考开发手册设计新表情包。
  8. 论坛备份数据结构有变动，建议进行一次备份。
  9. 注意检查 install.php 是否存在，若存在，删之。
  10. 升级完成

从 BMForum Plus! 升级说明
返回顶部
注：从 BMForum Plus! 升级前请确保已经成功安装了本版本

首先将 add-on 下的plus2datium.php放置于 Datium! 版目录下，然后用适当编辑器（如 Emeditor）打开文件，设置文件头的以下内容：

以下是代码:
$pluspostdir="C:/Apache/database/post/"; // Plus! 版帖子目录
$plususerdir="C:/Apache/database/user/"; // Plus! 版用户目录
$plusdatadir="C:/Apache/htdocs/bmb/datafile/"; // Plus! 版 datafile 目录


然后进入论坛管理中心(MySQL版本)，删除所有版块、分类，接着删掉自己的管理帐户
在浏览器执行 plus2datium.php。按照步骤一步一步操作即可。
