系统需求
返回顶部
要运行 BMForum，请确保论坛处于以下最低需求或更高的环境中：
  • PHP 环境: PHP 5.3.0 或更高
  • Web 环境: Apache, IIS, Nginx 等服务器
  • 数据环境: MySQL 5.4
  • 磁盘环境: 最低 50MB

推荐配置：
  • PHP 环境: PHP 7.0.0
  • Web 环境: Nginx
  • 数据环境: MariaDB 10.0
  • 磁盘环境: 1 GB

安装说明
返回顶部
  1. 打开压缩包，将 bmb 目录所有文件按照原来路径上传到服务器你新建的目录下(例如bmb)目录下。
  2. 设置权限，请根据服务器类型选择。
    1. Unix/Linux/FreeBSD 等 *nix 类服务器设置权限：(注:部分空间可能设置777后出错,请设置为755)
      • 目录 datafile
        目录内的文件和文件夹设置为 777
      • 目录 tmp
        目录内的文件和文件夹设置为 777
      • 目录 upload
        目录内的文件和文件夹设置为 777
      • 目录 upload/usravatars
        目录内的文件和文件夹设置为 777
      • 目录 datafile/cache
        目录内的文件和文件夹设置为 777
      • 目录 datafile/cache/levels
        目录内的文件和文件夹设置为 777
      • 目录 datafile/cache/themes
        目录内的文件和文件夹设置为 777
    2. Windows 类服务器（如IIS），一般不用设置，若无法使用，请联系管理员打开 IIS 进程权限。

  3. 上传完文件之后，打开您的浏览器，输入 install.php 文件的 URL 运行。 (例如：http://www.yourdomain.com/bmb/install.php) 输入正确详细的资料即可，输入前请仔细看清楚每个 选项前面的说明。
  4. 设置完毕后，程序会自动删除 install.php 文件的(如果无法自动删除的话，请利用 FTP 软件登陆后手动 删除)，并进入管理中心，你可以在管理面板中建立论坛，设置参数
  5. 在"常规选项"中设置所有的空白变量。
  6. 设置用户组
  7. 管理中心的设置结束后，就可以运行 index.php 来看你的论坛啦 :D
最后重复提示一次： 检查服务器上是不是还存在 install.php 和 install 目录， 如果有，请务必删除，否则非常危险，切记切记！
